My Creative Process

All of my UI/UX and design projects are based off of a practiced formula to get the result that I am looking for

1

Define the Problem

Ideas come from all over the place and regardles of how it happened, it's time to start executing. In this first step I will try to understand what problem are we trying to solve and gather all the requirements to move forward with the project.

2

Ideation

This is my favourite and most creative part of the whole process because we will start to create the wireframe from the UI library and drafting the web design. Everything is done with the goal and vision: to improve the functionality of the user interface.

3

Implementation and Release

After the design has been approved, I will help and work closely with other development team members to make sure that it will be executed accordingly. I'll always be ready to add more details and annotations to the wireframes, and if it doesn't feel right after being implemented to the real code, we can tweak the design to make it work.

4

Feedback and Improvement

It's been a long journey to get through from the idea to the production phase. However, we need to know whether our products and features are well received by the end users. This can be done by tracking usage via metrics or we can interview our users about it. This way after every project, we will always get better at designing!
